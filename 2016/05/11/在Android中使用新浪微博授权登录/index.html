在Android中使用新浪微博授权登录

发表于 2016-05-11   |  

新浪微博 Android SDK 为开发者提供了 Oauth2.0授权认证，并集成SSO登录功能，使第三方应用无需了解复杂的验证机制即可迚行授权登录操作,并提供微博分享功能,第三方应用可直接通过微博客户端迚行分享。
本文将总结在Android开发中，如何使用Weibo SDK进行SSO授权登录。

前提

  • 本实例基于 weibo_andorid_sdk on Github 中的 example。仅包含使用 SSO＋Web 方式授权登录的部分。
  • 在weibo_andorid_sdk on Github 中包含 Weibo SDK 文档，有更加详细的内容。
  • 本文程序可以从Github: WeiboLoginDemo中获得。
  • 需要集成 Weibo SDK 的android应用名称为：WeiboLoginDemo
  • 创建的微博开发平台应用名称为： LoginDemo4Chuanl

应用注册

创建应用

  • 在微博开发平台中选择移动应用
  • 选择移动接入
  • 填写应用名称，选择应用平台（android），点击创建，完成创建过程。

应用信息

在创建完成之后，将进入应用基本信息表单。

获取Android 签名

其中需要注意的是 Android包名 以及 Android签名.
Android 包名：填写自己应用的package name.
Android 签名：使用 Weibo SDK 提供的签名工具生成。

Clone weibo_android_sdk

从Github上clone weibo_android_sdk.

其中包含文档，demo，签名工具（app_signatures.apk）等。

安装签名工具
获取Android签名

在手机中打开刚刚安装的“MD5签名生成器”，输入应用的包名，并点击生成。

Note: 在生成签名前，必须已经安装”WeiboLoginDemo”应用。同时”WeiboLoginDemo”必须已经签名，如果不签名，会导致程序可以运行，但是无法授权的问题。

应用高级信息

编辑 授权回调页 与 取消授权回调页，都设置为 https://api.weibo.com/oauth2/default.html.

Studio 集成 Weibo SDK

WeiboLoginDemo 将使用 SSO+Web授权 方式，即如果手机上安装了微博客户端会默认发起SSO授权，反之则进行Web授权。
