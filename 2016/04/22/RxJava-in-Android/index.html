What is RxJava

官方Github定义：

RxJava – Reactive Extensions for the JVM – a library for composing asynchronous and event-based programs using observable sequences for the Java VM.

RxJava是一个JVM库，通过这个库可以使用观察序列来编写异步的，基于事件的程序。

过滤Observables

  • filter()

  • take(int N)
    只发送序列前N个。

  • takeLast(int N)
    只发送序列最后N个。
    Note: 只能用在一个完整的，有限大小的序列。

  • distinct()
    去掉序列中的重复项。
    Note: 只能用在一个完整的，有限大小的序列；由于需要记录每一项，用于判断是否重复，因此需要注意内存大小。

－ distinctUntilChanged()
只有序列中数据变化时才发送数据。
